1、基础入门-概念名词

域名

什么是域名？

例如：浏览器地址栏上 www.baidu.com

域名在哪里注册？

可以在万网（阿里云）、腾讯云里面注册

什么是二级域名？多级域名？

顶级域名，又称一级域名，常见的有”.com”、”.org”、”.net”、”.cn”等。

二级域名就是在一级域名前再加一级，如”baidu.com”。

二级域名以上级别的域名，统称为子域名，不在“注册域名”的范畴中。

可以设置解析域名地址，也就是将域名解析到对应IP地址上。

域名发现对于安全测试意义？

多级域名给渗透测试多了一个突破口，一种可能性。

分站可能与主站解析地址一致。

比如当www开头网站没有突破口，可以从jw域名寻找突破。

DNS

什么是DNS？

域名系统（服务）协议，主要用于域名与IP地址的相互转换。

本地HOSTS与DNS的关系？

hosts文件路径：

比如说在windows上设置www.yuyapeng.cn 解析到1.2.3.4

DNS如何解析域名地址

CDN是什么？与DNS的关系？

即内容分发网络，是一个缓存节点技术。

CDN其实并不是为了防止安全攻击，是为了减轻网站的服务器压力。

例如：ping youku.com

使用超级ping，从全国各地ping，DNS解析显示IP地址不同

常见的DNS安全攻击有哪些？

DNS劫持，DNS投毒，DDos攻击，DNS查询嗅探器

脚本语言

常见的脚本语言有哪些？

asp php aspx jsp javaweb pl py cgi等

不同脚本类型与安全漏洞的关系？

不同脚本，容易产生的安全漏洞类型不同。

严谨的脚本语言，产生漏洞几率小；

简单的脚本语言，产生漏洞几率大。

漏洞挖掘代码审计与脚本类型的关系？

代码审计必须要掌握脚本语言的使用，才能对源码进行漏洞挖掘。

后门

什么是后门？有哪些后门？

攻击者渗透成功后留下后门文件，便于下次进入进行操作。webshell，服务器后门，EXE后门，系统后门，内核后门

后门在安全测试中的实际意义？

便于下次进入
相当于一个通道，便于直接操作进入网站，服务器。

关于后门需要了解哪些？（玩法、免杀）

玩法：有各种各样的后门。根据目的选取不同的玩法，不同的后门。

比如说可以隐藏在软件下面，或者隐藏在网站上，或者直接控制。

免杀（必须要会）：后门容易被一些安全软件（火绒，360）查杀，所以要学免杀。

WEB

WEB的组成架构模型？

网站源码

操作系统：windows linux
中间件（搭建平台）：apache iis tomcat nginx等
数据库：acess mysql mssql oracle sybase db2 postsql等

ps：中间件就是为网站提供服务的，可以理解为杯子。喝水要有杯子，有各种各样的杯子

架构漏洞安全测试简要介绍？

web网站->数据库权限->权限提升-> 内网

为什么要从WEB层面为主？

不管是APP，PC软件，实质上都能以WEB作为突破口进入
